# Smart Shelf Life

A Machine Learning Approach to Texture Stability in Climate-Sensitive Foods

## Overview

Food wastage due to spoilage is a major global challenge.

This project introduces LightFusion, a lightweight multi-modal deep learning framework that combines:

- Food image analysis
- Temperature monitoring
- Humidity monitoring
- Storage duration analysis

to predict:

- Freshness Category
- Remaining Shelf Life (RSL)

## Features

- Deep Learning-based freshness prediction
- MobileNetV2 feature extraction
- Metadata fusion
- Remaining shelf-life estimation
- FastAPI deployment
- Explainable AI using Grad-CAM

## Technologies

- Python
- TensorFlow
- Keras
- OpenCV
- FastAPI
- NumPy
- Pandas

## Results

- Classification Accuracy: 87.5%
- R² Score: 0.83
